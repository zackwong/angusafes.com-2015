1. To open the door for the first time, please use the outer battery box and follow the steps below. After opening the door, please fit the batteries in the inner battery box and take out the outer battery box for normal operations.

2.To lock the door by user code

Close the door, input 3 to 6 digits code at your option, and confirm it by pressing the key “#”, the door will be locked automatically, and the code is successfully changed. You can also press the key “#” directly to lock the door without changing the code.

3.To open the door by user code

Input your code( factory setting code is 123456), and confirm it by pressing the key “#”, the door will be opened automatically.

4.To open the door by master code

Input the master code(factory setting code is 888888), and confirm it by pressing the key “#”, the door will be opened automatically in 5 seconds.

5.To change master code

When the door is locked, input the master code, and confirm it by pressing the key “#”, press the key “#” again in 5 seconds, input new 3 to 6 digits master code and confirm it by press the key “#”, “done” displays as indicates the master code is successfully changed.

6. In case you forget the code, you can open the door by using the override key instead of inputting the code. After opening the door, please lock the door by using the key, the door will be opened automatically, and the master code will go back to factory setting code “888888”.

7. Press the key “*” to activate the code hiding function before inputting the code.

8.Three times wrong try starts a 5-minute freezing period

9.Press the key “*” to delete the wrong input.

10.Setting Time：

When the door is open, input **200110, “date” displays,

when “ye” displays, please input year,

when “on” displays, please input month,

when “da” displays, please input day,

when “ho” displays, please input hour

when “in” displays, please input minute

when “se” displays, please input second

then “GOOD” displays as indicates the time has been successfully set.

11.Audit trailing:

105 pieces of opening records can be read via the screen.

When the door is open, input **200118, “record” displays, then the opening record displays by 3 screens, the interval is 3 seconds:

For example:

1.the first screen: 001 U means the No. of opening record is 001, opening method is U

2.the second screen: 120101 means the opening date is Jan., 1st, 2012

3.the third screen: 080808 means the opening time is 08:08:08

4.opening method:

U---opened by user code, C----by master code, b---opened by CEU, E---opened by key

5. Press “4” for next record, press “6” for last record.

12. Low battery warning

When the door is locked, press the key “#”, “lo-bat” displays as indicates low battery, please change the batteries as soon as possible. You can open the door by using the outer battery box when low battery.
