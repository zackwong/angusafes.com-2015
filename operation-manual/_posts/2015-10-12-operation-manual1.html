1.Pull the drawer to the position as the figure and fit the batteries in the battery box.

2.To open.

Input your code( factory setting code is 123), and confirm it by pressing “#”, “OPENED” displays, now pull the drawer to open.

3.To lock

Push the drawer in to the end, the drawer will be locked automatically.

4.To change the code

Enter **666# original code #new code #new code, “DONE” displays, as indicate the code has been successfully changed.

5.In case you forget the code, you can open the door by using the override key instead of inputting the code. The code will turn back to factory setting code “123”.please follow step 3 to change your code.

6.Press “*” to activate the code hiding function before inputting the code.

7.Three times wrong try starts a 5-minute freezing period

8.Press “*” to delete the wrong input.

9.Low battery warning

When operating, “LO-BATTE” displays as indicates low battery, please change the batteries as soon as possible.
