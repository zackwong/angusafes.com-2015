1. To open the door by CEU:

Pls note, pls use this function when the serial code of the CEU and the safe are the same.

A, Turn the CEU on, “GOOD” displays.

B, Input CEU management code (factory setting CEU management code is 6666) and confirm it by pressing “#”.

C, Connect the CEU and the safe by USB. Press any key except “#” and “*” on the safe panel to awake the safe.

D, Input the opening code (factory setting opening code is 8888) and confirm it by pressing “#”.

E, The door will be opened atomatically.

2, To change CEU management code:

Press the key “*” two times, “PROG” displays, enter original CEU management code*1, “NEW-C1” displays, then enter 4 digits new CEU management code #, “DONE” displays, as indicates the CEU management code has been successfully changed.

3, To change opening code:

Press the key “*” two times, “PROG” displays, enter original opening code*2, “NEW-C2” displays, then enter 4 digits new opening code #, “DONE” displays as indicates the opening code has been successfully changed.

4, MRC(management reset code) function:

Press the key “*” two times, “PROG” displays, enter 6080*0, “RESET” displays as indicates the operation is finished successfully, and the CEU management code turn back to factory setting “6666”,opening code turn back to factory setting “8888”.

5, To Read and print opening records:

Please note: please use the CEU to open the door, then follow the steps below:

A, Connect the CEU and the safe. Press any key except “#” and “*” on the safe panel to awake the safe.

B, Press the key “*” two times. “PROG”displays.

C, Enter 8668*8, “DONE” displays as indicates the CEU is reading the opening records from the safes, meanwhile the safes displays “SEND”; the CEU displays “DONE” as indicates the CEU has finished reading the opening records.

D, Install the software in your computer.

E, Connect the CEU and the converter, then connect the converter and the computer.

F, Use the software to read the opening and print the records.

6, To change the serial code of CEU:

Press the key “*” two time, “PROG” displays, enter 5665*5, then input the new serial code, and confirm it by pressing the key “#”, “DONE” displays as indicates the serial code has been successfully changed.

7, To change the serial code of the safe:

A, Connect the CEU and the safe. Press any key except “#” and “*” on the safe panel to awake the safe.

B, Enter 8090*9, “OLD” displays,

C, Enter original serial No. of the safe (factory setting No. is 0000), and confirm it by pressing the key “#”. “NEW” displays.

C, Input new 4 digits serial code of the safe and confirm it by pressing the key “#”, “DONE” displays as indicates the serial code of the safe changed successfully.

8, Renting function

A, To set renting time.

Connect the CEU and the safe, Press any key except “#” and “*” on the safe panel to awake the safe.

press the key “*” two times, “PROG” displays, enter 8090 *4, then input year and month( for example:1210 means Oct., 2012) and confirm it by pressing the key “#”, the screen turns to next one, input day and hour(for example:0101 means Jan. 1st), and confirm it by pressing the key “#”, “DONE” displays, as indicates the renting function has successfully set. And from now to the setting time, the safe can’t be opened by user code, but the safe still can be opened by management code or CEU.

B, To close renting function:

Connect the CEU and the safe, Press any key except “#” and “*” on the safe panel to awake the safe.

press the key “*” two times, “PROG” displays, enter 8090*5, “DONE”, as indicates the renting function has been successfully closed.
