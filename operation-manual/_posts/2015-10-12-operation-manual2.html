Functions:

1. Code can be set at 3 to 8 digits.

2. Five user codes(factory setting code is empty) provided, each user code can be used separately.

3. One management code provided(factory setting management code is 12345678)

4. Time delay opening function provided(factory setting time delay period is 00)

5. Opening mode can be set at your choice(factory setting mode is single).

6. Management Reset Code(MRC) provided.

7. Entry of four consecutive invalid code starts a 5-minute freezing period


Opening mode setting:

1. Single mode(to use one user code to open the lock): Enter **158#management code#1#, the green lamp lights, as indicate the setting is successful.

2. Dual mode(to use two user codes to open the lock): Enter **158#management code#2#, the green lamp lights, as indicates the setting is successful.

Please note: in dual mode, the lock can also be opened by using management code only.


Time delay function setting:

The delay period could be set from 0 to 59 minutes.

Setting: Enter **159#management code#00-59#, the green lamp lights as indicates the setting is successful.

00-----lock opened directly

01-----lock opened in 1 minute.

02-----lock opened in 2 minute.

03-----lock opened in 3 minute.

And so on

59-----lock opened in 59 minute

lease note: Time delay function is only for user code. the lock will always be opened without delay when using the management code.

Setting user code:

Factory setting user code is empty. Setting the user code by management code: Enter **333#management code # user No.# 3 to 8 digits user code#3 to 8 digits user code#, the green lamp lights as indicates the code is successfully set.

User No. can be 1,2,3,4,5.

Please note: If you want to change the user code after setting the user code, please use the user code to change it, please see below.

Changing user code:

Enter **444 # original user code # user No. of the original code # new 3 to 8 digits user code # new 3 to 8 digits user code#, the green lamp lights as indicates the code is successfully changed.

User No. can be 1,2,3,4,5.

For example, if you have set a code for user 1, and if the code for user 1 is 123. Now you want to change the code of user 1 from "123" to "456", please enter **444# 123#1#456#456#, now the code has been changed to "456"


Deleting user code by management code:

Enter **666#management code#user No.#user No.#, the green lamp lights as indicates the code is successfully deleted.


Changing management code:

Enter**888#original management code# new 3 to 8 digits management code# new 3 to 8 digits management code#, the green lamp lights as indicates the code is successfully changed.


Opening when time delay period is 00:

Single opening mode: Enter user code#, the green lamp lights, please open the door within 5 seconds.

Double opening mode: Enter first user code# and second user code#(any 2 user codes will work), the green lamp lights, please open the door within 5 seconds.


Opening when time delay period is 1 to 59 minutes:

After your correct user code is input and confirmed by #, the time delay function is activated. After the time delay period, the lock will alarm for 30 seconds, after the alarm, please open the door within 5 second.


Opening by using management code:

Enter management code #, the green lamp lights, please open the door within 5 seconds.


Low battery warning:

After opening the door, the red lamp lights and the alarm sounds as indicate low battery, please change the battery as soon as possible.


MRC(Management Reset Code) operation:

Use MRC: Enter *#*#*#MRC#, the red, green and yellow lamp will all light, as indicates MRC has worked successfully, and the lock system turns back to the factory setting status.
