1. To open the door by CEU:

Pls note, pls use this function when the serial code of the CEU and the safe are the same.

A, Turn the CEU on, “GOOD” displays.

B, Input CEU management code (factory setting CEU management code is 1111) and confirm it by pressing “#”.

C, Connect the CEU and the safe by USB.

D, Input the opening code (factory setting opening code is 2222) and confirm it by pressing “#”.

E, The door will be opened atomatically.

2, To change CEU management code:

Press the key “*” two times, “PROG” displays, enter original CEU management code*1, “NEW-C1” displays, then enter 4 digits new CEU management code #, “DONE” displays, as indicates the CEU management code has been successfully changed.

3, To change opening code:

Press the key “*” two times, “PROG” displays, enter original opening code*2, “NEW-C2” displays, then enter 4 digits new opening code #, “DONE” displays as indicates the opening code has been successfully changed.

4, MRC(management reset code) function:

Press the key “*” two times, “PROG” displays, enter 6208*9, “RESET” displays as indicates the operation is finished successfully, and the CEU management code turn back to factory setting “1111”,opening code turn back to factory setting “2222”.

5, To Read and print opening records:

A, Connect the CEU and the safe.

B, Press the key “*” two times. “PROG”displays.

C, Enter 5968*8, “READ” displays as indicates the CEU is reading the opening records from the safes, meanwhile the safes displays “SEND”; the CEU displays “DONE” as indicates the CEU has finished reading the opening records.

D, Install the software in your computer.

E, Connect the CEU and the converter, then connect the converter and the computer.

F, Use the software to read the opening and print the records.

6, To change the serial code of CEU:

Press the key “*” two time, “PROG” displays, enter 9207*5, “NEW-SO” displays, then input the new serial code, and confirm it by pressing the key “#”, “DONE” displays as indicates the serial code has been successfully changed.

7, To change the serial code of the safe:

A, Press the key “*” two times, “PROG” displays, enter 9027*6, the CEU turns to OVERRIDE mode.

B, Connect the CEU and the safe.

C, Input new serial code of the safe and confirm it by pressing the key “#”, “DONE” displays as indicates the serial code of the safe changed successfully.

D, If you want to set the serial code of other safes as the same code, press # directly when the CEU is in the OVERRIDE mode.

E, Turn off the CEU, after 3 seconds, re-turn on the CEU to quit from OVERRIDE mode.

Please note: the factory setting serial code of the CEU and the safe are the same “0000”.
